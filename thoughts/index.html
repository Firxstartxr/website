Hackathon AI Chatbot Challenge – Powered by NoCode-X

2025-03-13

This Hackaton was organized by Howest University. The goal was to create a chatbot using a no-code platform. The teams were assigned randomly and the main goal was to help companies, which provided their data for the project, help build a chatbot to automate routine tasks. The real strugle was to feed the data to the agent. I took part mostly in this but unfortunately it did not succeed. We could not configure an option on the platform to read data directly from PDF files, so I decided to create a small dataset by extracting text using a Pythong script. I don't think it is the best option but it was the only idea I could come up during a hackaton. Our team did not win and I think this is fair. We did not bring a lot of team effort, mostly we were doing tasks on our own and someone from the team did not contribute almost anything. If I were given more time and more possibilities to work with (I did not really enjoy no-code), I believe I would be able to come up with a better solution.

2025-02-11
